About This Project

This project is a collaboration between Todd Appleby and Su-Yee Lee, two neuroscientists.

Inspired by choose-your-own adventure books, online personality tests and neural circuits -

we wanted to create a multimedia project that was greater than the sum of its parts.

The website is constructed of linked web pages with connected paths informed by neural circuit motifs, e.g. recurrence, parallel pathways, feedforward/back.

We hope to make more stories and multimedia content in the future. This is the first.
